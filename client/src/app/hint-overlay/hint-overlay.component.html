A shape is selected
  • Escape to deselect
  • Delete to delete
  • Click-and-drag to move
  • Control-click to open formatting
  • Click other node in creation sidebar to replace
An edge is selected
  • Escape to deselect
  • Delete to delete
  • Click-and-drag on any circles to move them
  • Control-click to open formatting
  • Click other node in creation sidebar to replace
  • Double click to add labels
Nothing is selected
  • Click something to select it
  • Shift click-and-drag to select multiple things
  • Click-and-drag and edge from the sidebar to add it
Edge Creation Mode
  • Click and hold a green circle to select the start
  • Hold and release on a green circle to select the end
  • Press Escape to stop creating edges.
