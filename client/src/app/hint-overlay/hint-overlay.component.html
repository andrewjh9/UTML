A shape is selected
  • Escape to deselect
  • Delete to delete
  • Click-and-drag to move
  • Click-and-drag the red circles to resize
  • Double-click to edit the text
  • Control-double-click to open formatting
An edge is selected
  • Escape to deselect
  • Delete to delete
  • Click-and-drag on any circles to move them
  • Control-double-click to open formatting
  • Double click to add labels close to the middle start or end.
Nothing is selected
  • Click something to select it
  • Shift click-and-drag to select multiple things
  • Click-and-drag and edge from the sidebar to add it
A label is selected
  • Escape to deselect
  • Delete to delete
  • Click-and-drag to move it
  • Double click to edit the text
Edge Creation Mode
  • Click and hold a orange circle to select the start
  • Hold and release on a green circle to select the end
  • Press Escape to stop creating edges.
Editing Text
  • Escape or Control+Enter to stop.
  • Enter for a new line
  • Type \alpha for greek letters and math symbols.
Multiple things are selected
  • Escape or click elsewhere to deselect
  • Control click to add elements to the selection
  • Delete to delete everything
